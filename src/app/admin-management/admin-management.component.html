Espace gestion

Cet espace permet différentes fonctionnalités de gestion de l'association, il a pour but d'être modifié selon les besoins des différents mandats BDA

Mettre à jour les cotisations

A partir d'un fichier Excel comprenant les bons champs, le site mettra à jour les status de cotisation de tous les comptes créés qui figurent dans le fichier.
Cela ne dispense pas d'aller checker dans la base de données ! (https://phpmyadmin.alwaysdata.com/)
Les champs requis sont sensible à la casse, il y en a 3 :

  • Nom
  • Promotion (format : P + N° promo, ex P21)
  • Cotisant BDA (valeurs acceptées : Oui / Non)
